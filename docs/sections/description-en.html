A use-case

Figure 2 shows an instantiation of the ontology. It represents a criterion in the WELL v2 standard. The instantiation could be linked to linked building data using multiple links. The bco:Credit class could be linked to a feature of interest (such as :RoomX). Parameters could be linked to properties of these features of interests.

If the standardization body would change the objective values, :RoomX's HumidityControl assessment would also be updated automatically.
