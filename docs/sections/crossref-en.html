has partop back to ToC or Object Property ToC

IRI: https://alexdonkers.github.io/bao#hasPart

Relationship between parts and sub parts of a standard. Being the superclass of multiple part-of relationships, bao:hasPart could be used to simply query multiple part-of relationships.
Example
ao:hasPart is a transitive property, implying that if a bao:Standard -bao:hasPart-> bao:Category, and bao:Category -bao:hasPart-> bao:Credit, then reasoners would infer bao:Standard -bao:hasPart-> bao:Credit.

has characteristics: asymmetric, transitive
